Enabling communities for quality control management is simple, with this module, you can effortlessly monitor, review, and manage processes to ensure your server runs at its best.

## Configuration

<Info>This module contains elements from the Customization Module.</Info>

To configure quality control commands, execute `/Config > Configure Quality Control.` Next, select the channel for quality control submissions to be sent. Use the second menu to select the roles you consider Quality Control. Lastly, click the green `Enable Quality Control` button.
